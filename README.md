# Diagrammi di Programmazione

Questo repository contiene i diagrammi di flusso relativi agli esercizi di programmazione svolti durante il corso.  
I diagrammi sono stati realizzati con **Miro** e rappresentano la logica dei primi esercizi introduttivi.

---

## 📁 Struttura del repository

- **esercizio1.png.png**  
  Diagramma di flusso dell’esercizio 1.

- **esercizio2.png.png**  
  Diagramma di flusso dell’esercizio 2.

- **/screenshots/esercizio3.png**  
  Diagramma di flusso dell’esercizio 3, organizzato nella cartella dedicata agli screenshot.

---

## 📝 Descrizione degli esercizi

### 🔹 Esercizio 1  
Diagramma che rappresenta la logica di base di un semplice algoritmo introduttivo.

### 🔹 Esercizio 2  
Diagramma che estende la logica del primo esercizio con condizioni e flussi decisionali.

### 🔹 Esercizio 3  
Diagramma che illustra il processo “Come fare il caffè”, utilizzato come esercizio pratico per imparare a costruire diagrammi di flusso.
